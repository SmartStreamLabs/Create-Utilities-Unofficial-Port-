# Create Utilities (Unofficial Port)
Create Utilities (Unofficial Port) is a NeoForge 1.21.1 port of the open-source Create addon originally created by Duquee_.

This port is maintained by SmartStreamLabs for:
- Minecraft `1.21.1`
- NeoForge `21.1.x`
- Create `6.0.9`

The project keeps the original mod id `createutilities`, preserves the original MIT license, and retains attribution to the original project and author.

## Notes
- This repository is an unofficial port, not the original upstream project.
- Create is consumed as an external Gradle dependency and is not bundled into this project.
